## 目录结构🌳

```
src              # 源代码
├─api            # 所有请求
├─assets         #字体等静态资源
├─components     # 全局公用组件
├─filter         # 全局 filter
├─router         # 路由
├─store          # 全局 store管理 
└─views          # 页面级直接
│  App.vue       # 入口页面
│  main.js       # 入口文件
   

```
